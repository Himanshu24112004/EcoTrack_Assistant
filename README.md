# EcoTrack Assistant 🌱

## Overview

EcoTrack Assistant is an AI-powered web application designed to help individuals understand, track, and reduce their carbon footprint through personalized insights and actionable sustainability recommendations.

The platform analyzes daily lifestyle habits such as transportation, household energy usage, food consumption, and travel behavior to estimate environmental impact and provide customized improvement suggestions.

---

## Problem Statement

Design a solution that helps individuals understand, track, and reduce their carbon footprint through simple actions and personalized insights.

EcoTrack Assistant addresses this challenge by transforming environmental data into practical recommendations that users can immediately apply in their daily lives.

## How It Works

1. User enters lifestyle information.
2. The system estimates carbon emissions.
3. Emission categories are analyzed.
4. Personalized recommendations are generated.
5. Users track progress through the dashboard.
